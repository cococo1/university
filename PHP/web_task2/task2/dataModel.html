Short Description

Users' type might be either "author" or "respondent". Questions' type might be: text question, single choice question and multiple choice question. Users table has as a foreign key the user which created the survey. It is a one-to-many relationship - an author can have many surveys, but each survey has only one author. We can see similar realtionships between Surveys and Questions tables, Users-Questions and Questions-Answers.
